# Github backup
Backup my Github repositories to my Google Drive. Use python and relative APIs.
## Prerequisite
Install Github and Git APIs.
```
pip install GitPython
pip install PyGithub
```
## Usage
```
python github-backup.py [user_account] [password]
```
